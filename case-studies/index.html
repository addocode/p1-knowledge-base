Cases

Purpose

Case studies preserve context without forcing future decisions to relive entire chats. Each dossier stores thesis, evidence, risk, outcome and lessons.

Key takeaway. This section is designed as an operational reference, not a motivational note. Every claim should eventually map to a document, observation or calculation.

Coding system

PL-GDN-001 refers to the first Gdańsk case. IT-GOA-001 is reserved for the future Genova case.

Outcome logic

A case can be successful even with no purchase if it improves future decision quality.

Reading mode

Each case should read like an internal investment memo, not a travel diary.
